# Vault 内笔记与链接示例

## 典型 vault 结构（快消工作流）

```
obsidian vault/
  个人IP知识库/
    Apify快消趋势报告2026.md
    中国快消趋势报告2026.md
    数据集-TikTok趋势Top100-2026.md
    两地快消趋势合并报告2026.md    ← 本工作流新建
    快消市场机会解读2026.md        ← 本工作流新建
    选题库/
    ...
```

## 合并报告开头示例（Obsidian 版）

```markdown
# 2026 年两地年轻人快消趋势合并报告（海外 + 中国大陆）

**海外来源**：[[Apify快消趋势报告2026]]（TikTok 美国 Top100、Facebook 搜索）
**中国来源**：[[中国快消趋势报告2026]]（今日热榜 API：抖音 50 条、B站 100 条）
**商业机会解读**：→ [[快消市场机会解读2026]]
```

## 源报告附录表新增行示例（Apify）

```markdown
| **两地合并报告** | 海外+中国统一词频与可视化 | [[两地快消趋势合并报告2026]] |
| **商业机会解读** | 品类机会、卖点/内容模板、可执行清单 | [[快消市场机会解读2026]] |
```

## 中国报告正文新增句示例

```markdown
已合并版见 [[两地快消趋势合并报告2026]]，商业机会解读见 [[快消市场机会解读2026]]。
```

实际 vault 路径以项目为准（如 `obsidian vault/个人IP知识库/`）。
