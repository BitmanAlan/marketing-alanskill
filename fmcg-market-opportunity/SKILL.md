# 快消市场机会解读

依据**两地核心快消词频**（来自 [[fmcg-two-region-report-merge]] 或 [[fmcg-core-keywords-extract]]），按「市场聆听」逻辑（内容+评论双轨分析 → 从洞察到可执行动作），产出：  
1）**品类机会**（选品/上新优先级、卖点假设）；  
2）**卖点/痛点**（详情页与客服话术、风险词与 FAQ）；  
3）**内容模板**（选题/脚本结构、投放建议）；  
4）**可执行清单**（优先级、责任人建议）；  
5）**过程与结果度量建议**。  
可与 [[xhs-douyin-market-listening]] 配合使用，本 Skill 侧重「趋势词 → 机会解读」，市场聆听 Skill 侧重「关键词矩阵 + 评论采集 + 聆听闭环」。

## 何时使用

- 已有两地合并报告或核心词表，需要「所以呢？该做什么？」的可执行解读。
- 需要产出选品建议、话术、内容模板、优先级清单供团队落地。

## 输入

- **必须**：两地最核心快消词（海外 5 类 + 中国 5 类 + 跨区共性），或合并报告全文。
- **可选**：平台表现简述（如 TikTok 18-24 占比、抖音/B站热榜话题原文），用于丰富机会描述。

## 输出结构（必须包含）

1. **核心词频与信号归纳（输入摘要）**  
   表格：地域 | 最核心 5 类词 | 主要平台表现；加「跨区共性」一句。

2. **品类机会 → 选品与上新优先级**  
   - 高优先级（两地均有强信号）：表列品类、机会描述、选品/上新建议、卖点假设。  
   - 中优先级（单侧强信号、可跨区复制）：同上。  
   - 内容与投放侧机会（如娱乐/节日借势）：机会描述 + 动作。

3. **卖点/痛点 → 产品、话术与客服**  
   - 卖点建议（可写入详情页与投放话术），按品类列点。  
   - 痛点与风险词（建议纳入客服 FAQ 与舆情监控），按品类列点。

4. **内容模板 → 选题与投放策略**  
   - 可复用选题/脚本结构表：模板名、适用品类、结构要点、平台侧重。  
   - 投放与自营建议（按平台/品类）。

5. **可执行清单（优先级与责任人建议）**  
   表列：序号、动作、类型、优先级（P0/P1/P2）、责任人建议。

6. **过程与结果度量建议**  
   过程度量（如关键词覆盖率、分析周期）、结果度量（如声量、转化、复购）、闭环（如周复盘—月迭代）。

## 有效做法 vs 避免

| 有效做法 | 避免 |
|----------|------|
| 每条结论对应到动作（选品、话术、内容、舆情） | 只给洞察不给「谁在什么时间做什么」 |
| 优先级与责任人建议明确（P0/P1/P2） | 清单过长无优先级 |
| 与合并报告中的核心词表对齐，新词可后续纳入 | 解读与词表脱节 |
