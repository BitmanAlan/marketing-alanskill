# 一键推送到 GitHub（你只需做 1 步）

## 第 1 步：复制你的 Token

1. 打开：https://github.com/settings/tokens  
2. 若还没有 Token：点 **Generate new token (classic)**，勾选 **repo**，生成后**复制**那串（以 `ghp_` 开头）。  
3. 若已有 Token：直接复制。

---

## 第 2 步：在终端执行这一条命令（把 Token 贴进去）

打开终端，**复制下面整行**，把其中的 **`这里粘贴你的Token`** 替换成你刚复制的 Token（不要留空格），然后粘贴到终端里，回车：

```bash
cd /Users/limingxuan/dev/Trendradar/marketing-alanskill-repo && git push https://BitmanAlan:这里粘贴你的Token@github.com/BitmanAlan/marketing-alanskill.git main
```

例如你的 Token 是 `ghp_abcd1234xyz`，那整行就变成：

```bash
cd /Users/limingxuan/dev/Trendradar/marketing-alanskill-repo && git push https://BitmanAlan:ghp_abcd1234xyz@github.com/BitmanAlan/marketing-alanskill.git main
```

---

## 推送成功后

- 打开 https://github.com/BitmanAlan/marketing-alanskill 就能看到所有文件。  
- **安全提醒**：Token 会出现在终端历史里，用完后可到 https://github.com/settings/tokens 把该 Token 删掉，再重新生成一个以后用。
