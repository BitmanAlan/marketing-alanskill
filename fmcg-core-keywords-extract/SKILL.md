# 两地核心快消词频提取

从「海外」与「中国大陆」两份快消趋势报告（或合并报告）中，提取**与快消直接相关、可做品类或营销对标**的核心词，产出：  
1）**核心快消词频表**（品类/场景 × 海外词 | 中国词 | 两地共性）；  
2）**两地最核心 N 类词**（海外 N 条、中国 N 条，建议 N=5）；  
3）**跨区共性**（3～5 条）。  
本步骤可为 [[fmcg-market-opportunity]] 提供稳定输入。

## 何时使用

- 已有两地报告或合并报告，需要显式抽出「用于市场分析」的词表。
- 合并报告已含词频表但需单独维护一份「核心词清单」供多文档引用。

## 输入

- 海外报告（平台、数据范围、与快消相关的关键词/话题/标签）。
- 中国报告（同上）。
- 或：已生成的合并报告（含核心词频表与最核心 5 类）。

## 输出

1. **核心快消词频表**  
   表头：品类/场景 | 海外核心词（TikTok/Facebook） | 中国核心词（抖音/B站） | 两地共性。  
   品类维度建议：美妆、健康/生活方式、宠物、食品/特产/餐饮、饮料、母婴、节日/春节、娱乐/借势。

2. **两地最核心词（用于市场分析）**  
   - 海外：5 类，每类「关键词/标签 + 一句话说明」。  
   - 中国：5 类，同上。  
   - 跨区共性：3～5 条。

3. **可选**：Markdown 或 JSON 格式的「仅核心词」清单，便于脚本或下游 Skill 消费。

## 有效做法 vs 避免

| 有效做法 | 避免 |
|----------|------|
| 只保留「可做品类/营销对标」的词，过滤纯娱乐/时事 | 把热榜所有词都当核心词 |
| 两地同一品类下无对应词时写「—」或「本次未显性」 | 留空或编造 |
| 最核心 N 类与词频表品类一致，便于追溯 | 词频表与最核心词脱节 |

## 与其他 Skill 的关系

- **上游**：[[fmcg-two-region-report-merge]]（合并报告内可已含本步结果；本 Skill 可单独执行以产出「仅词表」）。  
- **下游**：[[fmcg-market-opportunity]] 必须基于本输出中的「两地最核心词」做品类机会与可执行清单。  
- 可与 [[xhs-douyin-market-listening]] 的关键词矩阵配合：本 Skill 产出「趋势侧核心词」，市场聆听 Skill 产出「监控矩阵」与评论分析。
