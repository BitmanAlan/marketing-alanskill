# 核心快消词频表 — 字段与品类维度

## 词频表表头

| 列名 | 说明 |
|------|------|
| 品类/场景 | 统一维度，见下表 |
| 海外核心词（TikTok/Facebook） | 该品类下海外报告中出现的关键词、话题、hashtag |
| 中国核心词（抖音/B站） | 该品类下中国报告中出现的热词/话题 |

## 建议品类/场景维度

| 品类/场景 | 说明 |
|-----------|------|
| 美妆 | 彩妆、护肤、美妆博主、借势美妆等 |
| 健康/生活方式 | 健康生活、饮食、运动、居家、零售体验等 |
| 宠物 | 宠物用品、清洁、宠物食品等 |

| 饮料 | 品牌体验、限定口味、自助/打卡等 |

若某侧无对应词，填「—」或「本次热榜未显性」。

## 两地最核心 N 类词格式

- **海外最核心 5 类**：每条为「**标签/关键词**（具体词）— 一句话说明（含占比/场景等）」。
- **中国最核心 5 类**：同上。
- **跨区共性**：3～5 条短语，如「美妆」「健康与饮食」「节日与场景消费」「娱乐借势」。
