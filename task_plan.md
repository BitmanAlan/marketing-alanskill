# Task Plan: Marketing-AlanSkill 工作流梳理与多 Skill 创建

## Goal

基于「两地快消趋势合并 → 可视化 → 核心词频 → 市场机会解读 → Obsidian 发布与双向链接」的完整操作路径，在本项目内创建 `marketing-alanskill` 文件夹，并拆解为多个可复用的 Skill，每个 Skill 独立文件夹，含优质 references、assets、scripts 等。

| 1 | 创建 task_plan / findings / progress（planning-with-files 规范） | complete |

## Decisions

- **Skill 拆分**：按操作步骤拆为 5 个 Skill（两地报告合并、可视化、核心词频提取、市场机会解读、Obsidian 发布与双向链接）。
- **References**：每个 Skill 的 `references/` 下放置工作流说明、输出示例、模板等，便于复现优质结果。
- **命名**：文件夹用英文 kebab-case（如 `fmcg-two-region-report-merge`），SKILL 标题与描述用中英双语。
